## Purpose
A healthy and respectful collaborative environment is key to the success of an open-source project. This code of conduct outlines expectations for contributors and the steps to report violations.

## Scope
This code of conduct applies to all contributors within the project spaces, including issue discussions, pull requests, and all project-related online/offline interactions.

## Expected Behavior
Contributors are expected to:
- Be welcoming and patient toward newcomers.
- Use inclusive language and be respectful of differing viewpoints.
- Provide constructive feedback during collaborations.
- Comply with project maintainersâ€™ decisions.

## Unacceptable Behavior
Unacceptable behaviors include:
- Disrespectful, aggressive, or discriminatory comments.
- Flaming or trolling other contributors.
- Publishing private information about others without their consent.
- Other conduct which violates professional standards.

## Consequences of Unacceptable Behavior
Violations of this code may lead to:
1. Verbal warning from the maintainer team.
2. Temporary or permanent ban from contributions.

## Reporting Issues
To report a code of conduct violation:
1. Contact project maintainer(s): [Ammmaarrr](https://github.com/Ammmaarrr)
2. Provide relevant information about the violation.

Maintainers will handle reports confidentially. Please help us foster a collaborative and respectful project environment.
